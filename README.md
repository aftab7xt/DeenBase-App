# DeenBase

> Your complete Islamic companion for Quran, Hadith, and daily Dhikr.

DeenBase is a free, open-source Android app built for Muslims who want a clean, distraction-free experience for reading Quran, exploring Hadith, and maintaining daily remembrance of Allah.

No ads. No clutter. Just content.

---

## ✨ Features

### 📖 Quran
- Read all 114 Surahs with English and Urdu translation
- Adjustable Arabic font size and style
- Set a daily reading goal and track your progress
- Beautiful edge-to-edge reading experience

### 📚 Hadith
- 35,000+ authentic Hadiths from 7 major collections
  - Sahih al-Bukhari
  - Sahih Muslim
  - Abu Dawud
  - Tirmidhi
  - Ibn Majah
  - Nasai
  - Muwatta Malik
- Browse by Book -> Chapter -> Hadith
- Full-text search across the entire database
- Grade displayed for each Hadith (Sahih, Hasan, Da'if)

### 📿 Dhikr
- Morning and Evening Adhkar
- Subhanallah x100 Tasbih counter with haptic feedback
- Beautiful animated UI

### 🔔 Notifications
- Gentle reminders for your daily Quran goal
- Morning and Evening Dhikr reminders
- Fully customizable notification times

### 🎨 Design
- Material 3 Expressive design system
- Dark and Light theme support
- Smooth animations throughout

## 🛠 Tech Stack

- **Language:** Kotlin
- **UI:** Jetpack Compose + Material 3 Expressive
- **Database:** Turso (libSQL cloud SQLite)
- **Networking:** OkHttp
- **Navigation:** Jetpack Navigation Compose
- **Min SDK:** Android 8.0 (API 26)

---

## 📌 Notes

- Internet connection is required for Quran and Hadith (data is fetched from the cloud)
- The keystore used for signing is kept private, do not submit PRs that modify signing config
- `local.properties` is gitignored and contains private API credentials, you will need to supply your own Turso database to build from source

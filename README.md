# DeenBase

A free, open-source Islamic Android app for Quran, Hadith, and daily Dhikr. Clean interface, no ads, no clutter.

## Features

- **Quran** - Read all 114 Surahs with English/Urdu translation, daily goal tracking
- **Hadith** - 35,000+ hadiths from Bukhari, Muslim, Abu Dawud, Tirmidhi, Ibn Majah, Nasai and Muwatta Malik. Browse by book/chapter or search
- **Dhikr** - Morning and Evening Adhkar, Tasbih counter
- **Notifications** - Customizable reminders for Quran and Dhikr
- **Design** - Material 3 Expressive, dark/light theme

## Download

Latest APK on the [Releases](https://github.com/aftab7xt/DeenBase-App/releases) page.

## Tech

Kotlin, Jetpack Compose, Material 3, Turso (libSQL), OkHttp

## Notes

- Requires internet connection
- `local.properties` is gitignored, you will need your own Turso database to build from source

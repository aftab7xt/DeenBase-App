# DeenBase

A free, open-source Islamic Android app built for Muslims who want a clean, distraction-free experience. Read Quran, explore Hadith, and maintain daily Dhikr — all in one place.

No ads. No clutter. Just content.

---

## Features

### 📖 Quran
- Read all 114 Surahs with English and Urdu translation
- Adjustable Arabic font size and style
- Set a daily reading goal and track your progress

### 📚 Hadith
- 35,000+ authentic Hadiths from 7 major collections
  - Sahih al-Bukhari, Sahih Muslim, Abu Dawud, Tirmidhi, Ibn Majah, Nasai, Muwatta Malik
- Browse by Book -> Chapter -> Hadith
- Full-text search across the entire database
- Grade shown for each Hadith (Sahih, Hasan, Da'if)

### 📿 Dhikr and Tasbih
- Morning and Evening Adhkar
- Tasbih Counter

### 🔔 Notifications
- Daily Quran goal reminders
- Morning and Evening Dhikr reminders
- Fully customizable notification times

### 🎨 Design
- Material 3 Expressive
- Dark and Light theme
- Smooth animations throughout

## 🛠 Tech Stack

- **Language:** Kotlin
- **UI:** Jetpack Compose + Material 3 Expressive
- **Database:** Turso (libSQL cloud SQLite)
- **Networking:** OkHttp
- **Min SDK:** Android 8.0 (API 26)
